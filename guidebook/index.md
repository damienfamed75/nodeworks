# Nodeworks

A programmable logistics and automation network. Build networks of small connection-based nodes and
program them from a terminal using Lua scripts.

## Getting Started

- [Broadcasting networks](broadcasting-network.md) — pair a broadcast antenna and a receiver antenna to extend a network across distance, dimensions, or both.

## Lua API Reference

- [network](lua-api/network.md) — live network access: queries, routing, inserts, callbacks.

The Scripting Terminal's editor hovers any documented identifier with its signature and
short description; pressing **G** while hovering opens the relevant page here.

See [docs/authoring.md](https://github.com/damienfamed75/nodeworks/blob/main/docs/authoring.md)
for how to add to these docs.
